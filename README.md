# Abba Api

1.- clonar el proyecto
2.- ``` yarn install ```
3.- Clonar el archivo ``` .env.template ```y renombrar a ``` .env ```
4.- cambiar las variables de entorno
5.- levantar la base de datos (container)
```
docker-compose up -d
```
6.- Levantar: ``` yarn start:dev ```
